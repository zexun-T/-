模板引擎art-template的使用步骤

1、下载art-template模板引擎库文件，并在HTML页面中引入库文件

https://raw.githubusercontent.com/aui/art-template/master/lib/template-web.js

<script src='./js/template-web.js'></script>

2、准备art-template模板

<script id='tpl' type="text/html"> //需要加入type="text/html"
    <div class='box'></div>
</script>

3、告诉模板引擎将哪一个模板和哪个数据进行拼接

<script>
    var html = template('tpl',{username:'zs',age:'20',sex:'男'})
</script>

4、将拼接号的html字符串添加到页面中

<div id='container'></div>  //html页面中先准备容器
document.getElementById('container').innerHTML = html

5、通过模板语法告诉模板引擎，数据和html字符串要如何拼接

<script id='tpl' type="text/html">
    <div class='box'>{{username}} {{age}} {{sex}}</div>
</script>

整体大概思路就是：传值>>>拼接>>>插入页面

 <script src="./template-web.js"></script>
<div id='container'></div>
<script id='tpl' type="text/html">
    <div class='box'>{{username}} {{age}} {{sex}}</div>  //拼接
</script>
<script>
    var html = template('tpl',{username:'zs',age:'20',sex:'男'})  //传值
    document.getElementById('container').innerHTML = html  //插入
</script>
