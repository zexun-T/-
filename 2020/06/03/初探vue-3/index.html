Vue.js 学习笔记

<template>

<template> 元素可以当做一不可见的包裹元素，当想要同时渲染多个元素的时候，可以用<template>包裹。

v-if条件渲染

v-if用以表示元素是否渲染的条件判断。

<template v-if="ifseen">
  <h1>Title</h1>
  <p>Paragraph 1</p>
  <p>Paragraph 2</p>
</template>

当实例中，ifseen为`true时,下面三条元素都会被渲染出来，v-if还可以是表达式判断语句：

<template v-if="Math.random() > 0.5">
  <h1>Title</h1>
  <p>Paragraph 1</p>
  <p>Paragraph 2</p>
</template>

当判断0~1的随机数大于0.5的时候，下面三条元素都会被渲染出来。

条件判断少不了v-else，v-else-if：（v-else必须放在判断的最后）

<div v-if="type === 'A'">
  A
</div>
<div v-else-if="type === 'B'">
  B
</div>
<div v-else-if="type === 'C'">
  C
</div>
<div v-else>
  Not A/B/C
</div>

这将会根据，实例中type的值判断渲染哪一部分元素

v-for列表渲染

v-for可以是遍历，对象、数组、数字。

当遍历对象时，有三个参数：value、name、index；分别表示该对象内的属性值，对象内的属性名，和对象内该属性的索引值

<ul id="example-1">
  <li v-for="value in object">
    {{value}}
  </li>
</ul>
<script>
    new Vue({
  		el: '#example-1',
  		data: {
    		object: {
     		title: '初探vue(3)',
			date: '2020-06-03 23:34:03',
			tags: 'vue框架学习'
    		}
  		}
		})
</script>

<ul id="example-1">
  <li v-for="(value,name,index) in object">
    {{index}}、{{name}}:{{value}}
  </li>
</ul>
<script>
    new Vue({
  		el: '#example-1',
  		data: {
    		object: {
     		title: '初探vue(3)',
			date: '2020-06-03 23:34:03',
			tags: 'vue框架学习'
    		}
  		}
		})
</script>

遍历对象，跟遍历数组，当索引值有多少个时，就会创建多少个元素，就像这个object里面有三个索引值，因此会创建三个<li>

而数组也有两个参数，一个是value、一个是index，数组有更多的更新操作

还有一种是数字的遍历

<div id='only'>
  <span v-for="n in 10">{{n}} </span>
</div>
<script>
    new Vue({
        el :'#only'
    })
</script>

这样会创建10个span标签，内容是12345678910

数组更新处理
<div id='only'>
	<ul>
		<template v-for="n in evenNumbers"></template>  <!-- 12334459 但是不会被渲染出来，完成一个排序后筛选--> 
		<li v-for="n in lastNumbers" role="presentation">{{ n }}</li> <!--1233 -->
    	<!-- <span>{{numbers}}</span> -->  <!-- sort函数对原先数组进行了改变-->
	</ul>
    	<!-- <li v-for="n in evenNumbers"></li> -->  <!--filter函数不会对数组进行改变，返回的是一个新数组-->
</div>
<script>
    var vm = new Vue({
            el: '#only',
            data: {
                numbers: [9, 4, 3, 1, 2, 3, 4, 5]
            },
            computed: {
                evenNumbers: function () {
                    return this.numbers.sort(function (a, b) {  //对原数组做出了改变
                        return a - b;
                    })
                },
                lastNumbers: function () {
                    return this.numbers.filter(function (number) {   //不会对原数组做出改变。
                        return number <= 3
                    })
                }

            }
        })
</script>

表单输入处理

表单处理重要的指令——

v-model

表单主要有文本、多行文本、复选框、单选按钮、选择框

v-model一开始读取初始值是在实例中的值，在表单中，v-model监听的是值的变化，值的变化导致实例的值变化，实例的值变化，导致渲染的文本变化，这就是文本的双向绑定

<div id="example-2">
  <select v-model="value">
    <option disabled value="">请选择你的英雄：</option>  <!--disabled为禁用选项。v-model 表达式的初始值未能匹配任何选项，<select> 元素将被渲染为“未选中”状态。在 iOS 中，这会使用户无法选择第一个选项。因为这样的情况下，iOS 不会触发 change 事件。因此，提供一个值为空的禁用选项。-->
    <option>亚瑟</option>
    <option>妲己</option>
    <option>宫本武藏</option>
  </select>
  <span>你选择的英雄为: {{ value }}</span>
</div>
<script>
    new Vue({
        el: '#example-2',
        data: {
            value: ''
        }
    })

</script>
