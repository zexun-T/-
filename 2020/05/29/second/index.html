学习前端编程，首先从HTML开始

HTML入门

html是一种超文本标记语言（HyperText Markup Language），用于创建网页的标记语言。

HTML标签：

实例：
<html lang="en">  //根标签，lang是属性名, ""引号里面为属性值(告诉搜索引擎网站的内容)
<head>
    <meta charset="UTF-8"> //设置字符集
    <meta name="viewport">
    <title>我的博客</title>  //浏览器窗口的标题    
</head>
<body>
    Hello,world!
</body>//主体内容，展示给用户的
</html>

其他写在body里面的处理文本的标签：

1、<p></p> 段落成段展示:

<p>燕子去了，有再来的时候；杨柳枯了，有再青的时候；桃花谢了,有再开的时候。但是，聪明的，你告诉我，我们的日子为什么一去不复返呢？——是有人偷了他们罢：那是谁？又藏在何处呢？是他们自己逃走了罢——如今又到了哪里呢？</p>
<p>我不知道他们给了我多少日子，但我的手确乎是渐渐空虚了。在默默里算着，八千多日子已经从我手中溜去，像针尖上一滴水滴在大海里，我的日子滴在时间的流里，没有声音，也没有影子。我不禁头涔涔而泪潸潸了。</p>

1、ul li无序列表

    <li>1</li>

    <li>3</li>

2、img图片标签

<img stc=" " alt=" "  title=" ">

src=’’ 可引用：
1、网上的url
2、本地的绝对地址
3、本地的相对地址
alt=’’(记述图片，地址错误时展示文字信息)

3、a标签 <a></a>

<a href=" "  target="blank">www.baidu.com</a>

href（超文本引用）的属性值为链接、电话号码、邮件地址、某个锚点id值 、协议限定符等
target表示另一个标签页面打开网页操作

1、放地址，实现超链接
2、可做锚点，回到顶部，从目录选择到内容
3、打电话发邮件
4、协议限定等（发起提示框等功能）

<div id='demo2' style="position:absolute;top:3500px;width:100px;height:100px;background-color: green;">
</div>
<a href="#demo2">find demo2</a>

4、form表单

<form action=" " method="get/post">  </form>

表格标签，发送数据的功能
组成元素：method=”get/post” （数据发送方式)
action=” “(发送给谁/发送上传数据)

组件：

（一）<input> 输入框; 供用户输入, 提交按钮

5、div与span

div跟span都是没有特性的标签，但是几乎所有元素都会创建div或者span标签，作为一个容器处理。
区别：div将文本和元素集中在一个块级元素中，故div标签为一个块级元素标签
span标签将文本和元素集中在一个内联元素中，故span标签标签为一个内联元素标签
ps:块级元素跟内联元素区别会在CSS文档中介绍

id特性跟class特性

每个HTML元素都可以附带一个class属性，class属性可以有多个值
每个HTML元素也都可以附带一个id属性，id属性有且只能有一个值，表示该元素的唯一标识。
两者的作用是为了方便利用css或javascript中选择并且操作标签

id属性 class属性
id属性有且只能有一个值，该元素的唯一标识 class属性可以有多个值
<div id = "only"> <div class = "demo1 demo2">
